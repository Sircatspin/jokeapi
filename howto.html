Joke Generator API Documentation

Base URL

http://5.161.225.129:5003

Endpoints

GET /joke

Retrieves a random joke from the joke database.

Response Format

JSON

Example Response

{
  
}

Usage

Retrieve a random joke:

  • Endpoint: /joke
  • Method: GET
  • Response: Returns a random joke from the jokes.txt file in JSON format.

Example Request

GET http://5.161.225.129:5003/joke

Example Response

{
 {"joke":"127. Why can't you give Elsa from Frozen a balloon? Because she will let it go!"}
}
